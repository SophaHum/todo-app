# Requirements:
- laravel 10,
-  React 18
-  Node 18+

# Setup
- git clone ......
- cd to project directory
- copy .env.example -> .env -> connect with your DB
- compoer install
- php artisan migrate
- php artisan migrate:fresh --seed
- npm install
- npm run dev && php artisan serve
